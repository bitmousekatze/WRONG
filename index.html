// 001 — The Concept

A Coin That Gets Smaller
As The World Gets Worse

Every verified war event creates a community burn target. Anyone holding $WRONG can burn toward it — sending tokens to a unique on-chain address permanently. Gone. Forever. The supply shrinks. The ledger grows.

We can't stop anything. But we can count it. And every count burns.

// The Math
10 bombs dropped in one day + 2 bridges destroyed + 3 civilians confirmed killed = 3,100,000 $WRONG burned. Permanently. On-chain.
01
War Does Something Bad
Community spots a verified event from official sources — DoD, UN OCHA, IAEA. Posted to Discord, voted in, confirmed. We add it to the ledger.
02
Target Gets Set
Civilian killed? 1,000,000 $WRONG target. Airstrike? 10,000. It scales with severity. The math is not complicated. The situation is.
03
You Burn. It's Permanent.
Send $WRONG to the event's unique address. It's gone. On-chain. Immutable. The ledger never forgets, even when everyone else does.
