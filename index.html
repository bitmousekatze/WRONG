// 001 — The Concept

A Protest In Code

$WRONG tracks the US–Iran military conflict in real time. Verified war events create community burn targets — anyone holding $WRONG can burn toward that target, removing tokens from circulation permanently, on-chain, forever.

This isn't one person's project. It's thousands of people choosing to make their protest permanent. Every burn is a statement. Every tx hash is evidence.

01
Event Verified
A war event is confirmed from an official source — DoD, UN OCHA, UN OHCHR, or IAEA. It gets added to the ledger with a unique reference code and a burn target.
02
Community Burns
Anyone holding $WRONG sends tokens to the burn address with the event ref code as memo. No approval. No minimum. Burn what you can.
03
Ledger Updates
Within ~60 seconds your wallet, amount, and date appear on the ledger under that event. Permanent. Verifiable on Solscan. No trust required.
// 002 — Community Burns

Burn Together

No whale required. No single person responsible. You burn what you can — 10,000 or 10,000,000. Every contribution is recorded on-chain. When enough people burn, the target is filled and the event is permanently marked.
